# Content Quality Checker - 内容质量检验器

## 概述

本技能是微课视频生产线的质量保障技能，负责对微课脚本和视频内容进行多维度的质量检验。通过自动化的内容质量评估和人工审核辅助，确保微课内容符合教学标准、知识准确、表达清晰、案例适切。

## 核心功能

### 1. 知识点准确性检验

**检验维度：**

| 检验项 | 检验方法 | 标准 | 自动化程度 |
|-------|---------|------|----------|
| 概念定义准确性 | 与权威资料比对 | 一致性≥95% | 半自动 |
| 术语使用规范性 | 术语库比对 | 规范术语使用率≥98% | 自动 |
| 数据引用准确性 | 数据源核查 | 数据准确率100% | 半自动 |
| 理论引用正确性 | 学者/理论库核对 | 引用准确率≥99% | 半自动 |
| 时间/地点准确性 | 事实核查 | 事实错误≤0 | 半自动 |

**检验规则库：**

```python
准确性检验规则 = {
    "概念定义": {
        "方法": "NLP实体识别+知识库比对",
        "标准": "核心概念定义与权威定义匹配度≥0.95",
        "处理": "低于阈值→标记疑似错误→人工复核"
    },
    "专业术语": {

        "标准": "术语使用100%符合规范",

        "处理": "无法溯源→删除或标注"数据待核实""
    }
}
```

### 2. 逻辑连贯性检验

**检验维度：**

| 检验项 | 检验方法 | 标准 | 自动化程度 |
|-------|---------|------|----------|
| 结构完整性 | 模板匹配 | 包含开场-主体-结尾 | 自动 |
| 段落衔接 | 过渡词检测 | 衔接词覆盖率≥80% | 自动 |
| 因果关系 | 逻辑链分析 | 因果链完整率≥90% | 自动 |
| 概念递进 | 知识图谱比对 | 递进关系符合认知规律 | 半自动 |
| 前后呼应 | 关键词回溯 | 首尾呼应度≥70% | 自动 |

**连贯性评分算法：**

```python
连贯性评分 = 函数(
    script_content,
    knowledge_graph
) -> {
    "结构得分": 0-100,
    "衔接得分": 0-100,
    "逻辑得分": 0-100,
    "总分": 加权总分
}

评分维度:
  - 结构得分: 权重0.3（开场10%+主体80%+结尾10%）
  - 衔接得分: 权重0.35（段落过渡、语义连贯）
  - 逻辑得分: 权重0.35（因果关系、推理正确性）
```

## 故障排除

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 检验超时 | 内容过长或规则库过大 | 分段检验或优化算法 |
| 误判专业术语 | 术语库不完整 | 扩展术语库 |
| 逻辑评分偏低 | 规则过于严格 | 调整阈值或增加例外 |
| 案例适切性评估不准 | 受众画像不准确 | 完善受众画像或人工介入 |

## 技术实现框架

```python
class ContentQualityChecker:
    def __init__(self):
        self.accuracy_checker = AccuracyChecker()
        self.coherence_analyzer = CoherenceAnalyzer()
        self.language_analyzer = LanguageAnalyzer()
        self.case_evaluator = CaseEvaluator()
        self.duration_checker = DurationChecker()
        self.report_generator = ReportGenerator()
    
    def check(
        self,
        content: dict,
        reference: dict,
        learner_profile: dict,
        options: dict = None
    ) -> QualityReport:
        # 1. 准确性检验
        accuracy_result = self.accuracy_checker.check(
            content, reference
        )
        
        # 2. 连贯性检验
        coherence_result = self.coherence_analyzer.analyze(
            content, reference.get('knowledge_graph')
        )
        
        # 3. 语言规范性检验
        language_result = self.language_analyzer.analyze(
            content, learner_profile
        )
        
        # 4. 案例适切性检验
        case_result = self.case_evaluator.evaluate(
            content, learner_profile
        )
        
        # 5. 时长合规性检验
        duration_result = self.duration_checker.check(
            content
        )
        
        # 6. 综合评分与报告
        report = self.report_generator.generate(
            accuracy_result,
            coherence_result,
            language_result,
            case_result,
            duration_result,
            options
        )
        
        return report
```
