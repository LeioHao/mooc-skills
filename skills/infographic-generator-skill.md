# Infographic Generator - 信息图生成器

## 概述

本技能是微课视频生产线的视觉内容增强技能，负责根据知识点内容和教学目标自动生成通俗化的示意图。通过智能化的图示类型推荐、视觉元素设计和风格适配，将抽象概念转化为直观易懂的图形呈现，提升学习者的理解效果。

## 核心功能

**图示类型详细定义：**

| 图示类型 | 适用场景 | 视觉特征 | 复杂度 |
|---------|---------|---------|-------|
| 概念图 | 展示概念及其关系 | 节点+连线+标签 | 低 |
| 关系图 | 展示多元素相互作用 | 圆形布局+箭头 | 中 |
| 流程图 | 展示步骤和决策 | 方框+菱形+箭头 | 低 |
| 时间轴 | 展示时间顺序和阶段 | 水平/垂直+里程碑 | 低 |
| 对比图 | 展示异同比较 | 左右对照/表格 | 低 |
| 因果图 | 展示因果链条 | 鱼骨图/因果链 | 中 |
| 模型图 | 展示系统结构 | 多层结构+交互 | 高 |
| 统计图 | 展示数据和趋势 | 柱状/饼图/折线 | 中 |

## 故障排除

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 图示过于复杂 | 知识点内容过多 | 应用简化策略，分拆为多图 |
| 布局不美观 | 自动布局算法局限 | 手动微调关键节点位置 |
| 与PPT风格不统一 | 缺乏模板参数 | 从ppt-template-selector获取配色 |
| 动画效果不佳 | 时间轴设计问题 | 简化动画，延长展示时间 |

## 技术实现框架

```python
class InfographicGenerator:
    def __init__(self):
        self.content_parser = ContentParser()
        self.type_selector = TypeSelector()
        self.layout_engine = LayoutEngine()
        self.style_applicator = StyleApplicator()
        self.animation_designer = AnimationDesigner()
    
    def generate(
        self,
        knowledge_point: dict,
        design_options: dict
    ) -> InfographicResult:
        # 1. 内容解析
        parsed = self.content_parser.parse(knowledge_point)
        
        # 2. 类型选择
        recommended_types = self.type_selector.recommend(
            parsed.type,
            parsed.relationships
        )
        
        # 3. 布局设计
        layout = self.layout_engine.generate(
            elements=parsed.elements,
            type=recommended_types[0],
            constraints=design_options.constraints
        )
        
        # 4. 风格应用
        styled = self.style_applicator.apply(
            layout=layout,
            style=design_options.style,
            colors=design_options.colors
        )
        
        # 5. 动画设计
        animation = self.animation_designer.design(
            infographic=styled,
            timeline=design_options.animation_timeline
        )
        
        return InfographicResult(
            elements=styled.elements,
            layout=styled.layout,
            animation=animation,
            export_format=design_options.format
        )
```
