# MOOC微课视频制作系统

微课视频生产线的模块化技能系统，实现从自然语言需求到MOOC认证视频的端到端自动化生产。

## 🎯 项目简介

本项目是一套完整的微课视频制作技能体系，包含13个核心技能，覆盖需求解析、内容构建、视觉设计和质量保障等全流程。通过模块化的技能编排和质量门禁控制，实现高效、高质量的系列微课视频生产。

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- Git
- 支持的操作系统: macOS, Linux, Windows

### 安装

```bash
# 克隆项目
git clone https://github.com/LeioHao/mooc-skills.git
cd mooc-skills

# 查看所有可用技能
ls skills/
```

### 使用示例

```javascript
// 导入技能
const CourseCreator = require('./skills/course-creator');
const NLPParser = require('./skills/natural-language-course-parser');

// 使用自然语言创建微课
const nlpParser = new NLPParser();
const courseRequirement = nlpParser.parse(
  "我想制作一门关于发展心理学的微课，面向年轻父母"
);
```

## 🎓 应用案例

### 案例一：发展心理学微课系列

**需求描述：**
```
课程名称：发展心理学
目标受众：年轻父母（0-6岁儿童家长）
课程要求：
- 单节微课5-12分钟
- 每节配2-3张通俗化示意图
- 重点结合生活案例讲解
```

**生产成果：**
- 6个模块，24节微课
- 每节包含完整脚本、PPT和视频
- 全部通过MOOC标准认证

### 案例二：职业技能培训系列

**需求描述：**
```
课程名称：Python数据分析
目标受众：职场数据分析师
课程要求：
- 单节微课10-15分钟
- 理论与实操结合
- 包含Jupyter Notebook演示
```

## 🔧 开发指南

### 添加新技能

1. 在 `skills/` 目录下创建技能文件
2. 编写技能文档（参考现有技能格式）
3. 实现技能代码逻辑
4. 在course-creator中注册技能

### 技能开发规范

- 遵循标准输入输出规范
- 实现完整的错误处理
- 提供详细的日志记录
- 编写单元测试
